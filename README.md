
# Overview

RAG-Based AI Teaching Assistant is an AI-powered learning tool that answers questions from educational videos using Retrieval-Augmented Generation (RAG). It converts video lectures into text, stores knowledge in a vector database, and provides accurate answers through an LLM.

# Process

1. Upload educational video lectures.
2. Convert video/audio into text transcript.
3. Split transcript into smaller chunks.
4. Generate embeddings for each chunk.
5. Store embeddings in vector database.
6. User asks questions related to lecture.
7. Retrieve relevant content from database.
8. LLM generates accurate context-based answers.
